# React-Webpack-Foundation Boilerplate

## Features
- Webpack building system
- Foundation 6

## Webpack Loaders
- URL
- File
- Styles: SCSS
- Imports (shimming modules)
- Babel for React and EcmaScript6

## Installing
`npm install`
`webpack`

## Running
run `node server.js` then check in your browser at http://localhost:3000
